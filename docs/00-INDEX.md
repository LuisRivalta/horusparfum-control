## Para a IA

Ao iniciar uma sessão de trabalho neste projeto:
1. Leia `HANDOFF_IA.md` para saber onde paramos
2. Consulte `PRD.md` para regras de negócio
3. Use `ARQUITETURA.md` para navegação de código
4. Ao finalizar, atualize `HANDOFF_IA.md` e adicione entrada em `LOGS.md`
